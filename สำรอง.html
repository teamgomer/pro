VPN CONFIG PANEL

เลือกใช้งานตามระบบปฏิบัติการของคุณ

🤖 Android (DARK TUNNEL)

📥 ดาวน์โหลด DARK TUNNEL

วิธีติดตั้ง Android:

1. คัดลอก Config > 2. เปิด DARK TUNNEL > 3. กดสามจุดขวาบน > 4. เลือก Config > 5. เลือก Import > 6. เลือก Clipboard > 7. กด OK แล้ว Connected

🍎 iOS (NPV TUNNEL)

📥 ดาวน์โหลด NPV TUNNEL

วิธีติดตั้ง iOS:

1. คัดลอก Config > 2. เปิด NPV TUNNEL > 3. กด Configs > 4. กด + > 5. กด Import From Clipboard > 6. กด SSH CONFIG > 7. กดชื่อเซิร์ฟเวอร์ให้เป็นสีฟ้า แล้ว Connected
